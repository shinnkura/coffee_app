## 特徴

-コーヒー豆の種類を一覧表示

-各コーヒー豆の詳細ページ

-コーヒー豆のカートボタンと説明

## 動作環境

・flutter 2.X
・Dart 2.X

## インストール方法

以下のコマンドを実行して、このリポジトリをクローンします：

```bash
git clone https://github.com/yourusername/flutter-coffee-app.git
```

Flutter SDK がインストールされていることを確認し、以下のコマンドを実行して依存関係を取得します：

```bash
flutter pub get
```

その後、以下のコマンドを実行してアプリケーションを起動します：

```bash
flutter run
```
